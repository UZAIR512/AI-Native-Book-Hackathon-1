## üöÄ Getting Started with the Blog

Setting up a blog in Docusaurus is easy ‚Äî you just add Markdown or MDX files inside the `blog` directory.

Each post can include:

- A `slug` for clean URLs  
- Authors from `authors.yaml`  
- Tags for categorization and filtering  
- MDX components for interactivity  
- Images stored next to the post folder  

For example, your blog post structure can look like:

- `2021-08-26-welcome.md`
- `2021-08-26-welcome/index.md`
